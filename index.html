Desenvolvimento de Soluções Tecnológicas para a IIGD

Durante minha atuação na Igreja Internacional da Graça de Deus (IIGD), desenvolvi e implementei diversas soluções tecnológicas para otimizar processos internos e melhorar a gestão de informações.

Portal Web para Prestação de Contas

Criei um portal web para que os pastores pudessem enviar prestações de contas, incluindo arrecadações e despesas das igrejas. A solução foi desenvolvida para atender à abrangência nacional da IIGD, proporcionando maior controle e transparência financeira.

Integrações via API e Webservice

Implementei webservices para facilitar a comunicação entre sistemas legados, garantindo interoperabilidade e eficiência na troca de dados. Além disso, realizei a integração com a Cielo (versões 1 e 3), permitindo transações seguras e automatizadas.

Painel Inteligente Integrado à Central Telefônica

Desenvolvi um Painel Inteligente totalmente integrado à central telefônica, utilizado pelo programa de TV Fala Amigo. Essa solução permitiu um atendimento mais ágil e eficiente, facilitando a interação entre a equipe do programa e os telespectadores.

Desenvolvimento de Soluções Tecnológicas para a Etalent

Fui convidado para participar do desenvolvimento de uma nova versão de um software para Web, uma oportunidade desafiadora onde pude superar minhas próprias limitações e aplicar toda a experiência adquirida ao longo da minha trajetória.

Minha atuação incluiu desenvolvimento, manutenção e suporte de aplicações essenciais para a plataforma, garantindo alto desempenho, escalabilidade e integração eficiente entre sistemas.

Principais Contribuições

Análise de Perfil Comportamental: Desenvolvimento de funcionalidades para avaliar e interpretar perfis comportamentais, fornecendo insights valiosos para gestão de talentos.

Loja Virtual: Implementação e manutenção de uma plataforma de e-commerce, garantindo uma experiência de compra intuitiva e segura.

Sistema de Gestão de Carreira (MyEtalent): Desenvolvimento e aprimoramento do sistema MyEtalent, auxiliando profissionais na gestão de suas carreiras com base em perfis comportamentais.

Integração VAGAS ETALENT: Implementação de integração entre o sistema Etalent e a plataforma VAGAS, otimizando processos de recrutamento e seleção.

Integração ao Sistema Etalent via WebService: Desenvolvimento de webservices para facilitar a comunicação entre sistemas e garantir a interoperabilidade dos dados.

Desenvolvimento de Soluções Tecnológicas para a Estado da Arte Informática e Tecnologia

Fui pioneiro na utilização e desenvolvimento de sistemas web na empresa, empregando tecnologias como ASP.NET, AJAX e XML para modernizar processos e otimizar o acompanhamento de atendimentos realizados via 0800. O sistema permitia monitorar todo o ciclo de atendimento, desde o primeiro contato até a realização da venda e o suporte pós-venda, garantindo maior controle e eficiência operacional.

Aplicativo de Consultas Gerenciais Multicamadas

Idealizei e desenvolvi um aplicativo multicamadas voltado para consultas gerenciais dinâmicas, permitindo que gerentes e diretores criassem suas próprias análises de forma rápida e visual, utilizando grids e gráficos interativos.

Benefícios da Solução:

  • Redução de custos com papel, tinta de impressora e tempo operacional.
  • Flexibilidade e autonomia para gestores na geração de relatórios personalizados.
  • Atualização periódica dos dados para garantir informações sempre atualizadas.

Tecnologias e Arquitetura Utilizadas:

  • Aplicativo Cliente ➝ WebService ➝ SQL Server
  • Banco de Dados: O SQL Server realizava a importação automática dos dados quatro vezes ao dia, integrando informações de sistemas legados em FoxPro.

Essa experiência consolidou minha atuação na modernização de processos empresariais, aplicando arquitetura multicamadas, integração de sistemas e desenvolvimento de soluções gerenciais inteligentes.
